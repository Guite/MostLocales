Einführung

ModuleStudio separiert Web-Anwendungen in technische Schichten:

Models
Datenschicht zur Definition der Datenbank-Strukturen und Behaviour.
Controller
Prozessschicht zur Definition von Anwendungsfällen und Benutzerinteraktion.
Views
Darstellungsschicht zur Spezifikation kosmetischer Aspekte.

DSL

Meta-Modell
Basis-Teil von ModuleStudio, definiert Konzepte und Constraints
Editoren
Die graphischen Editoren, die die GUI zur Modellierung repräsentieren
Verwendung allgemeiner Begriffe zur Modellierung von MVC-Anwendungen
Generator
Erstellt Quellcode aus gebauten Modellen
